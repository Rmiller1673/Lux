About the Lux device

This project aims to create a low cost, home automation illumination device that will turn any lamp or light source into a smart light source. Instead of replacing the light bulb or wall socket, we aim to replace the light socket itself, allowing the user to continue using the light bulbs they already have for their homes. In addition, we will have custom firmware, custom hub software to connect multiple smart sockets and this custom web application to interface with the devices.

The user of this device would connect their new Lux to their computer to run a simple setup program, allowing them to view specific information regarding their device, i.e. serial number, and edit network information such as the IP address and port number assigned to the hub (server) and the network ssid and password for the wireless network. The Lux would replace the socket the light bulb is placed in, and would need to be properly connected with the lamp. The lamp light socket will both power the Lux Wi-Fi light socket and the light bulb connected to the other Lux terminal. The user would be able to control particular lights within their house using this accompanying webpage.
